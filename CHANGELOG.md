#### 📋 現在の料金分岐ロジック（2026-03-20時点）

**料金表:**
| | 一般 | 中高生 |
|---|---|---|
| ビジター（初参加・非利用者） | ¥3,000 | ¥2,000 |
| 前回出場者 | ¥2,500 | ¥1,500 |
| Studio会員（スタジオ利用者） | ¥2,000 | ¥1,500 |

**分岐フロー:**
1. メンバーを「スタジオ利用者」と「それ以外」に分ける
2. スタジオ利用者 → 年齢区分に応じた Studio会員 価格（一般¥2,000 / 中高生¥1,500）
3. それ以外 → 年齢区分 × 前回出場の有無で単価決定
   - 前回出場=はい → 前回出場者価格（一般¥2,500 / 中高生¥1,500）
   - 前回出場=いいえ → ビジター価格（一般¥3,000 / 中高生¥2,000）
4. 各メンバーの単価 × 人数 を合算して合計金額を算出

**優先順位:** スタジオ利用者 > 前回出場者 > ビジター
（前回出場者かつスタジオ利用者の場合、スタジオ利用者価格が適用される）
